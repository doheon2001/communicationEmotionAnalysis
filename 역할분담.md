## [1단계] 공통 데이터 정리

- **도헌** — 감성 데이터 담당 (AIHub 감성 대화 말뭉치, K-MHaS)
- **문경** — 공격성 데이터 담당 (AIHub 텍스트 윤리검증 데이터, K-HATERS)
- **설하** — 팀 로그/통합 데이터 담당 (팀 대화 로그)

## [2단계] 전처리 코드 작성

- 전원 같이 진행
- 데이터 정제 / 토큰화 / 라벨 정리 / 학습용 형태 변환 담당
- 전처리 코드는 공통으로 관리

## [3단계] 모델 학습

- **도헌** — 감정 분석 모델 메인 담당
- **문경** — 공격성 분석 모델 메인 담당
- **설하** — 두 모델 보조 + 결과 분석 담당
- 감정 모델과 공격성 모델은 **분리 학습**

## [4단계] 평가 및 튜닝

- **도헌** — 감정 모델 평가/튜닝
- **문경** — 공격성 모델 평가/튜닝
- **설하** — 시각화 및 F1-score 정리 담당

## [5단계] 최종 통합

- **도헌** — 통합 작업 참여
- **문경** — 통합 작업 참여
- **설하** — 전체 시스템 메인 연결 및 최종 통합 담당
- 최종 단계에서 두 모델 결과를 하나로 연결

## 추가 메모
