실제로 쓰는 개념만** 골라 쉽게 풀어 설명합니다.

## ChaeOn에서 ML을 쓰는 곳

ChaeOn의 5개 기능 중 **ML 모델은 기능 1·2뿐**입니다.

- **기능 1 — 감정 분류**: 대화 텍스트 → 긍정/부정 수치
- **기능 2 — 공격성 탐지**: 대화 텍스트 → 공격성 여부/확률

기능 3·4·5는 모델이 아니라, 기능 1·2의 결과나 통계를 **집계·시각화**하는
계층입니다. 그래서 이 학습 문서는 주로 **기능 1·2를 만드는 데 필요한 ML 지식**에
집중합니다.
